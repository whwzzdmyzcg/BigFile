大文件求和
问题:有4个文件，每个文件1,0000,0000个32位整数，以空格分隔

对这四个文件中的整数分别求和。

输入:./a.out 1.in 2.in 3.in 4.in

输出:1.in xxxxxxxxxx2.in xxxxxxxxxx3.in xxxxxxxxxx4.in xxxxxxxxxx

注:a.out为可执行文件名,可任意。　

xxxxxxxxxx表示对应文件所求之和
